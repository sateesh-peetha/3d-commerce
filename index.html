Design System

A token-driven, block-based UI system for 3D commerce and manufacturing. Everything is a configurable block. All visuals are token-driven. All blocks are device-aware.

Governing Principles

Everything is a configurable UI block

All visuals are token-driven

All blocks are device-aware

All blocks are plugin-injectable
