# Contributing to 3D Commerce Platform

Thank you for your interest in contributing! We welcome pull requests from the community.

## Getting Started

1.  **Read the Docs**: Familiarize yourself with the [System Overview](docs/architecture/system-overview.md) and [Developer Guide](docs/getting-started/installation-guide.md).
2.  **Fork & Clone**: Fork the repository and clone it locally.
3.  **Install Dependencies**: Run `npm install`.

## Development Workflow

1.  **Create a Branch**: `git checkout -b feature/my-new-feature`
2.  **Make Changes**: Write code and tests.
3.  **Verify**: Run `npm run test:all` to ensure no regressions.
4.  **Commit**: Use conventional commits (e.g., `feat: add new plugin hook`).
5.  **Push**: Push to your fork and submit a Pull Request.

## Pull Request Guidelines

-   **Description**: Clearly explain the problem and solution.
-   **Tests**: Include tests for new functionality.
-   **Docs**: Update documentation if necessary.
-   **CI**: Ensure all status checks pass.

## Code of Conduct

Please be respectful and constructive in all interactions.
